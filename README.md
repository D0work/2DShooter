# Scarlet Dispute

A **bullet-hell shoot'em up inspired by Touhou Project**, featuring fast-paced combat, reactive difficulty, and challenging boss fights.

## 🎮 Gameplay
- **Movement**: `W` `A` `S` `D`
- **Combat**: Bullets follow the mouse position (no player rotation)
- **Boss Battles**:
  - Bosses grow stronger as their health decreases
  - Increased fire rate, tracking shots, and up to 4 summoned aides
  - Currently 1 level with 1 boss
- **Player Mechanics**:
  - Limited lives system
  - Dynamic HP bar for feedback
- **Difficulty Progression**:
  - Reactive gameplay that scales as damage is dealt

## 🆕 Latest Changes
